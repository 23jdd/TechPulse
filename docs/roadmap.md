# Roadmap

1. Phase 1: runnable MVP with RSS ingestion, MySQL, Bleve, mock AI, RAG, REST, and WebSocket.
2. Phase 2: move module calls to RabbitMQ and HTTP service clients.
3. Phase 3: production AI providers, hybrid search, embeddings, and conversation memory.
4. Phase 4: GitHub OAuth, OPML, favorites, and personalized prompts.
5. Phase 5: Kubernetes manifests, tracing, dashboards, CI/CD, and load tests.
