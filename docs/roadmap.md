# Roadmap

1. Phase 1: runnable MVP with RSS ingestion, MySQL, Bleve, mock AI, RAG, REST, and WebSocket.
2. Phase 2: independent HTTP services, RabbitMQ publisher/consumer, etcd registry, and distributed lock support.
3. Phase 3: OpenAI-compatible embeddings, hybrid search reranking, and conversation memory.
4. Phase 4: GitHub OAuth URL generation, OPML import/export, read later, reading history, favorites, personalized prompts, and daily reports.
5. Phase 5: Kubernetes starter manifests, Prometheus-ready endpoints, CI workflow, Docker validation, and benchmark coverage.
