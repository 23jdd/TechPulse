# 路线图

1. Phase 1：可运行 MVP，包含 RSS 采集、MySQL、Bleve、Mock AI、RAG、REST、WebSocket。
2. Phase 2：独立 HTTP 服务、RabbitMQ publisher/consumer、etcd registry、分布式锁。
3. Phase 3：OpenAI 兼容 embedding、混合搜索重排、会话记忆。
4. Phase 4：GitHub OAuth URL、OPML 导入导出、稍后读、阅读历史、收藏、个性化 prompt、日报。
5. Phase 5：Kubernetes 起步清单、Prometheus-ready endpoint、CI workflow、Docker 校验和 benchmark 覆盖。
