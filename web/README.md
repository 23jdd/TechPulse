# TechPulse Web App

[中文说明](README.zh-CN.md)

Open the user app after the gateway is running:

```text
http://localhost:8080/
http://localhost:8080/app
```

Login page:

```text
http://localhost:8080/login
```

Chinese UI:

```text
http://localhost:8080/login/zh
http://localhost:8080/zh
http://localhost:8080/app/zh
```

The app is a Tailwind-based user interface served by the gateway. It supports the core reading workflow:

- add RSS feeds
- test, enable, disable, and delete feeds
- fetch RSS and GitHub releases
- read article streams
- search articles
- view AI summaries
- mark read, save, read later, archive, and delete articles
- ask RAG questions with citations
- generate a daily report
