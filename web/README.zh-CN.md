# TechPulse Web Dashboard

gateway 启动后打开：

```text
http://localhost:8080/
http://localhost:8080/dashboard
```

中文页面：

```text
http://localhost:8080/zh
http://localhost:8080/dashboard/zh
```

Dashboard 是由 gateway 提供的 Tailwind 静态页面，用于演示核心工作流：

- 添加 RSS Feed
- 测试、启用、禁用、删除 Feed
- 触发抓取
- 查看文章列表
- 搜索文章
- 查看摘要
- 标记已读、收藏、归档、删除文章
- 带引用来源的 RAG 问答
- 生成每日技术日报
